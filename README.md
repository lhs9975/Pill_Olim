# 딥러닝을 활용한 알약 이미지 인식 앱(약올림)
__알약 이미지 인식 앱__


## 프로젝트 소개
__CNN을 활용하여 알약의 각인 및 모양 학습을 통해 정확한 알약 분류.__  
__생활에서 복용하는 알약에 대한 상세 정보 및 위험도를 사용자에게 안내.__  
__사용자 개인의 알약 일정 관리를 가능하게 함.__
